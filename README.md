# YOLO Labeller
Small, lightweight YOLO Labeling tool developed using `.NET Core 3.1 Windows Forms`.

## Installation
In the `Releases` tab, there are prebuilt binaries for `x86` and `x64` Windows platforms contained in the `Windows.zip` file.
